# SQL for QA Engineers 

This repository demonstrates SQL skills essential for Quality Assurance Engineers.

## What’s Covered

* Joins (INNER, LEFT, RIGHT)
* Aggregations (GROUP BY, HAVING)
* Subqueries
* Real-world testing scenarios

## Purpose

To showcase how SQL is used in real QA tasks such as:

* Data validation
* Backend testing
* Detecting data inconsistencies
* Verifying business logic

## Tools Used

* MySQL / PostgreSQL
* Sample e-commerce database

## Why This Matters

Strong SQL skills help QA engineers validate data accuracy and identify defects that UI testing cannot catch.
